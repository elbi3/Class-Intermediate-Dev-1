# Dynamic Toolbar Assignment

## Description

This project demonstrates dynamic rendering of React components from an array of data. The Toolbar component generates multiple AlertButton components using .map(). Each button displays a unique alert message when clicked, showing how to pass props and use children dynamically in React.

## How to Run the Project

1. Install dependencies:

```
npm install
```

2. Start the development server:

```
npm run dev
```

3. Open the local development URL shown in the terminal (usually `http://localhost:5173`) in your browser.
