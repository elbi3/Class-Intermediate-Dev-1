# Static Product List

This project:
- Demonstrates array iteration using `map()`
- Renders dynamic JSX from structured data
- Uses unique keys for React’s reconciliation process
- Separates concerns with a reusable component

## ▶️ How to Run the Project
1️⃣ Install dependencies
npm install

2️⃣ Start the development server
npm run dev

3️⃣ Open in browser
Vite will output a local URL, typically:
http://localhost:5173
